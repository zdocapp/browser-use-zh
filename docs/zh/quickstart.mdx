## 2. 选择您喜欢的 LLM

创建一个 `.env` 文件并添加您的 API 密钥。没有密钥？从[免费的 Gemini 密钥](https://aistudio.google.com/app/u/1/apikey?pli=1)开始。

<Tabs>
<Tab title="Mac/Linux">
```bash 创建 .env 文件
touch .env
```
</Tab>
<Tab title="Windows">
```cmd 创建 .env 文件
echo. > .env
```
</Tab>
</Tabs>

<Tabs>

```bash 将密钥添加到 .env 文件
GEMINI_API_KEY=
```
</Tab>

</Tab>

## 3. 运行你的第一个智能体

<Tabs>
<Tab title="谷歌">
```python agent.py
from browser_use import Agent, ChatGoogle
from dotenv import load_dotenv
import asyncio

load_dotenv()

async def main():
    llm = ChatGoogle(model="gemini-2.5-flash")
    task = "Find the number 1 post on Show HN"
    agent = Agent(task=task, llm=llm)
    await agent.run()

if __name__ == "__main__":
    asyncio.run(main())
```
</Tab>
