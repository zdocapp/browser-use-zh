## 使用 Azure OpenAI 的安全设置

企业级安全与 Azure OpenAI、数据隐私保护以及受限的浏览器访问。

```python
import asyncio
import os
from dotenv import load_dotenv
load_dotenv()
os.environ['ANONYMIZED_TELEMETRY'] = 'false'
from browser_use import Agent, BrowserProfile, ChatAzureOpenAI

# Azure OpenAI configuration
api_key = os.getenv('AZURE_OPENAI_KEY')
azure_endpoint = os.getenv('AZURE_OPENAI_ENDPOINT')
llm = ChatAzureOpenAI(model='gpt-4.1-mini', api_key=api_key, azure_endpoint=azure_endpoint)

# Secure browser configuration
browser_profile = BrowserProfile(
    allowed_domains=['*google.com', 'browser-use.com'], 
    enable_default_extensions=False
)

# Sensitive data filtering
sensitive_data = {'company_name': 'browser-use'}

# Create secure agent
agent = Agent(
    task='Find the founders of the sensitive company_name',
    llm=llm,
    browser_profile=browser_profile,
    sensitive_data=sensitive_data
)

async def main():
    await agent.run(max_steps=10)

asyncio.run(main())
```

## 安全特性

**Azure OpenAI:**

- 不用于训练 OpenAI 模型
- 不与其他客户共享  
- 完全托管在 Azure 内
- 30 天数据保留（或通过受限访问计划实现零保留）

**浏览器安全:**

- `allowed_domains`: 限制导航至受信任的站点
- `enable_default_extensions=False`: 禁用潜在危险的扩展
- `sensitive_data`: 从 LLM 输入中过滤敏感信息
