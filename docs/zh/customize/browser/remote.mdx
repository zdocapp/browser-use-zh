## 获取 CDP URL

### 云浏览器

从您喜欢的浏览器提供商（如 AnchorBorwser、HyperBrowser、BrowserBase、Steel.dev 等）获取 cdp url。

### 代理连接

```python

from browser_use import Agent, Browser, ChatOpenAI
from browser_use.browser import ProxySettings

browser = Browser(
        headless=False,
        proxy=ProxySettings(
            server="http://proxy-server:8080",
            username="proxy-user",
            password="proxy-pass"
        )
        cdp_url="http://remote-server:9222"
)


agent = Agent(
    task="Your task here",
    llm=ChatOpenAI(model='gpt-4.1-mini'),
    browser=browser,
)
```
