Browser Use Cloud API 定价包含两个组成部分：

1. **任务初始化成本**：每启动一个任务 $0.01
2. **任务步骤成本**：根据每个步骤使用的具体模型产生的额外费用

## 成本计算示例

例如，使用 GPT-4.1 执行一个 10 步骤的任务：

- 任务初始化：$0.01
- 10 步骤 x 每步骤 $0.03 = $0.30
- **总成本：$0.31**
