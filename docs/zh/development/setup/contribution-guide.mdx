## 使命

- 让开发者感到快乐
- 比人类完成更多点击操作
- 告诉你的计算机要做什么，它就会完成
- 让智能体更快更可靠

## 可以做什么？

- 这个领域发展迅速。我们每天都有10个新想法。让我们交流一下
- 浏览我们的 [GitHub Issues](https://github.com/browser-use/browser-use/issues)
- 查看我们在 [Discord](https://discord.gg/zXJJHtJf3k) 上最活跃的问题
- 在 [`#showcase-your-work`](https://discord.com/channels/1303749220842340412/1305549200678850642) 频道获取灵感

## 优秀的PR应具备什么？

1. 为什么我们需要这个PR？
2. 包含演示截图/gif
3. 确保PR通过所有CI测试
4. 保持PR专注于单一功能

## 如何参与？

1. Fork代码库
2. 为你的功能创建新分支
3. 提交PR

我们收到了大量Issue。如果需要更快的反馈，请随时通过评论定期提醒你的issues/PRs。
