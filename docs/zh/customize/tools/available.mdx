### 导航与浏览器控制

- **`search_google`** - 在 Google 中搜索查询
- **`go_to_url`** - 导航至 URL  
- **`go_back`** - 在浏览器历史中后退
- **`wait`** - 等待指定秒数

### 页面交互

- **`click_element_by_index`** - 按索引点击元素
- **`input_text`** - 在表单字段中输入文本
- **`upload_file_to_element`** - 上传文件至文件输入框
- **`scroll`** - 向上/向下滚动页面
- **`scroll_to_text`** - 滚动至页面特定文本
- **`send_keys`** - 发送特殊按键（回车、退出等）

### 标签页管理

- **`switch_tab`** - 在浏览器标签页间切换
- **`close_tab`** - 关闭浏览器标签页

### 内容提取

- **`extract_structured_data`** - 使用 LLM 从网页提取数据

### 表单控件

- **`get_dropdown_options`** - 获取下拉选项值
- **`select_dropdown_option`** - 选择下拉选项

### 文件操作

- **`write_file`** - 将内容写入文件
- **`read_file`** - 读取文件内容  
- **`replace_file_str`** - 替换文件中的文本

### 任务完成

- **`done`** - 完成任务（始终可用）
