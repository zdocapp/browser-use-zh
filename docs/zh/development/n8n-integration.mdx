# Browser Use n8n 集成

Browser Use 可以通过我们的社区节点与工作流自动化平台 [n8n](https://n8n.io) 集成。此集成允许您直接从 n8n 工作流触发浏览器自动化任务。

## 安装 n8n 社区节点

在 n8n 中安装 Browser Use 社区节点有几种方式：

### 使用 n8n Desktop 或 Cloud

1. 导航至 **设置 > 社区节点**
2. 点击 **安装**
3. 在 **名称** 字段中输入 `n8n-nodes-browser-use`
4. 点击 **安装**

### 使用自托管的 n8n 实例

在您的 n8n 安装目录中运行以下命令：

```bash
npm install n8n-nodes-browser-use
```

### 用于开发

如果您想要使用 n8n 节点进行开发：

1. 克隆代码库：
   ```bash
   git clone https://github.com/draphonix/n8n-nodes-browser-use.git
   ```
2. 安装依赖：
   ```bash
   cd n8n-nodes-browser-use
   npm install
   ```
3. 构建代码：
   ```bash
   npm run build
   ```
4. 链接到您的 n8n 安装：
   ```bash
   npm link
   ```
5. 在您的 n8n 安装目录中：
   ```bash
   npm link n8n-nodes-browser-use
   ```

## 设置 Browser Use Cloud API 凭据

要在 n8n 中使用 Browser Use 节点，您需要配置 API 凭据：

1. 在 [Browser Use Cloud](https://cloud.browser-use.com) 注册账户
2. 导航至设置或 API 部分
3. 生成或复制您的 API 密钥
4. 在 n8n 中创建新凭据：
   - 转到**凭据**标签页
   - 点击**创建新凭据**
   - 选择 **Browser Use Cloud API**
   - 输入您的 API 密钥
   - 保存凭据

## 使用 Browser Use 节点

安装完成后，您可以将 Browser Use 节点添加到工作流中：

1. 在工作流编辑器中，于节点面板搜索 "Browser Use"
2. 将节点添加到工作流
3. 设置凭据
4. 选择已保存的凭据
5. 选择操作：
   - **运行任务**：使用自然语言指令执行浏览器自动化任务
   - **获取任务**：检索任务详情
   - **获取任务状态**：检查任务执行状态
   - **暂停/恢复/停止任务**：控制运行中的任务
   - **获取任务媒体**：检索截图、视频或 PDF
   - **列出任务**：获取任务列表

### 示例：运行浏览器任务

以下是一个使用 Browser Use 节点运行浏览器任务的简单示例：

1. 将 Browser Use 节点添加到您的工作流中
2. 选择 "Run Task" 操作
3. 在 "Instructions" 字段中，输入您希望浏览器执行操作的自然语言描述，例如：
   ```
   访问 example.com，对首页进行截图，并提取所有主标题文本
   ```
4. 可选启用 "Save Browser Data" 以保留 cookies 和会话信息
5. 将节点连接到后续节点以处理结果

## 工作流示例

Browser Use n8n 节点支持多种自动化场景：

- **网页抓取**：按计划从网站提取数据
- **表单填写**：跨 Web 应用程序自动化数据录入
- **监控**：检查网站状态并捕获视觉证据
- **报告生成**：生成 Web 仪表板的 PDF 或截图
- **多步骤流程**：使用会话持久性将浏览器任务串联起来

## 故障排除

如果您遇到 Browser Use 节点的问题：

- 请验证您的 API 密钥是否有效且具有足够额度
- 检查您的指令是否清晰明确
- 对于复杂任务，考虑将其分解为多个步骤
- 参考 [Browser Use 文档](https://docs.browser-use.com) 了解指令最佳实践
